Senior Web Developer

Job Title

Senior Web Developer

Description

You'll bring years of hands-on experience to the table — leading technical decisions, setting standards for the team, and building robust, scalable web solutions. This is a role for someone who's been around the codebase a few times and knows exactly what good looks like.

Responsibilities

  • Architect and develop complex web applications.
  • Collaborate with designers to implement sleek, modern UI/UX.
  • Teach junior and trainee web developers.
  • Ensure code quality, performance, and security across all projects.

Required Qualifications

  • 5+ years of experience in full-stack web development.
  • Bachelor's degree in Computer Science or related field (or equivalent experience).
  • Proven track record of delivering high-quality, production-ready code.

Skills

  • Expertise in HTML5, CSS3, JavaScript (ES6+).
  • Deep knowledge of modern frameworks (React, Vue, or Angular).
  • Experience with backend technologies (Node.js, Python, or Ruby).
  • Familiarity with database architecture (SQL and NoSQL).
  • Proficiency with Git, CI/CD, and Agile methodologies.
  • Excellent communication and teamwork skills.

Wages

£26-£35/hour
£55,000-£70,000/year

Junior UI/UX Designer

Job Title

Junior UI/UX Designer

Description

We need someone to help us make the internet look a little bit better. You'll be working right alongside our developers to turn loose ideas and wireframes into interfaces that feel incredibly satisfying to use.

Responsibilities

  • Create wireframes, mockups, and prototypes for web applications.
  • Design modern, visually stunning user interfaces.
  • Conduct user research to improve usability.
  • Collaborate with developers to ensure accurate implementation.

Required Qualifications

  • 1-2 years of experience in UI/UX design (or strong portfolio).
  • Degree or bootcamp certification in Design or Human-Computer Interaction.
  • A keen eye for typography, color theory, and layout.

Skills

  • Proficiency in Figma, Sketch, or Adobe XD.
  • Basic understanding of HTML and CSS limitations.
  • Strong communication and presentation skills.
  • Ability to iterate rapidly based on feedback.

Wages

£17-£20/hour
£35,000-£42,000/year

Trainee Web Developer

Job Title

Trainee Web Developer

Description

You don't need to know everything yet, but you do need to be eager to learn. This role is all about rolling up your sleeves, writing real code, and figuring things out with the support of our senior developers. We'll help you improve your web development skills in a fast-paced but super supportive environment.

Responsibilities

  • Write clean, functional code (with guidance and code reviews).
  • Squash bugs and help optimize our existing web projects.
  • Pair program with our senior developers to learn the ropes.
  • Ask lots of questions and soak up knowledge like a sponge.

Required Qualifications

  • Certificate/Bootcamp Training in Web Development or in a related field
  • Must be at least 16 years old

Skills

  • Understanding of HTML, CSS, and JavaScript.
  • Basic understanding of frameworks and libraries.
  • Good communication skills.
  • Ability to work in a team
  • Eagerness to learn.

Wages

£8-£12/hour
£18,000-£20,000/year

Front-End Web Developer

Job Title

Front-End Web Developer

Description

We're looking for a Front-End Web Developer who takes real pride in what users see and feel. You'll be turning designs into fast, polished, and accessible web experiences — someone who obsesses over the details, knows their way around modern JavaScript frameworks, and understands that a great interface is about more than just making things look good.

Responsibilities

  • Build responsive, accessible, and performant user interfaces.
  • Translate designs and wireframes into clean, maintainable code.
  • Collaborate closely with designers and back-end developers.
  • Optimise pages for speed, SEO, and cross-browser compatibility.
  • Maintain and improve existing front-end codebases.

Required Qualifications

  • 2+ years of experience in front-end development.
  • Degree, bootcamp, or equivalent hands-on experience.
  • Strong portfolio demonstrating real shipped projects.
  • Solid understanding of web performance and accessibility standards.

Skills

  • Expertise in HTML5, CSS3, JavaScript (ES6+).
  • Proficiency with a modern framework such as React, Vue, or Svelte.
  • Experience with responsive design and CSS preprocessors (SASS/SCSS).
  • Familiarity with version control using Git.
  • Understanding of web accessibility (WCAG) and SEO best practices.
  • Ability to work from Figma or similar design tools.

Wages

£26-£35/hour
£55,000-£70,000/year

Back-End Web Developer

Job Title

Back-End Web Developer

Description

We're looking for a Back-End Web Developer who's comfortable in the engine room. You'll be building and maintaining the server-side logic, databases, and APIs that power our web applications — someone who values clean architecture, reliable performance, and code that other developers actually enjoy working with.

Responsibilities

  • Design and maintain server-side logic and RESTful APIs.
  • Manage and optimise databases (SQL and NoSQL).
  • Ensure application security, scalability, and performance.
  • Collaborate with front-end developers on API integration.
  • Write thorough documentation and maintain code quality.

Required Qualifications

  • 2+ years of back-end development experience.
  • Degree, bootcamp, or equivalent practical experience.
  • Solid understanding of databases and server architecture.
  • Experience with authentication, security, and data protection.

Skills

  • Proficiency in Node.js, Python, Ruby, or similar backend language.
  • Experience with databases such as MongoDB, PostgreSQL, or MySQL.
  • Strong understanding of RESTful API design and integration.
  • Familiarity with cloud platforms (AWS, GCP, or Azure).
  • Version control with Git and experience in CI/CD pipelines.
  • Understanding of web security best practices.

Wages

£17-£20/hour
£35,000-£42,000/year

Full-Stack Web Developer

Job Title

Full-Stack Web Developer

Description

We need a Full-Stack Web Developer who's equally at home in the browser and on the server. You'll be taking features from concept to completion — handling everything from slick front-end interfaces to solid back-end logic and database design. If you like variety, ownership, and shipping things end-to-end, this is the role for you.

Responsibilities

  • Develop and maintain both front-end and back-end features.
  • Design and consume RESTful APIs across the full stack.
  • Collaborate with designers and other developers on delivery.
  • Ensure performance, security, and reliability across projects.
  • Contribute to architectural decisions and code reviews.

Required Qualifications

  • 2+ years of full-stack development experience.
  • Degree, bootcamp, or strong portfolio of shipped projects.
  • Demonstrated ability to work across both front and back end.
  • Good problem-solving skills and attention to detail.

Skills

  • Proficiency in HTML5, CSS3, and JavaScript (ES6+).
  • Experience with a front-end framework such as React or Vue.
  • Back-end experience with Node.js, Express, or similar.
  • Database knowledge across SQL and NoSQL (e.g. MongoDB, PostgreSQL).
  • Confident with Git, REST APIs, and Agile workflows.
  • Ability to context-switch between front and back end effectively.

Wages

£20-£28/hour
£40,000-£55,000/year
